Projets

Système RAG avec n8n et Data.gouv

Stage de Master 1 - 2024-2025

Ce projet consistait à concevoir un système intelligent utilisant le RAG (Retrieval Augmented Generation) pour interroger des données publiques du site Data.gouv.fr.

Technologies utilisées : n8n (automatisation), Python, Bases de données vectorielles, Modèles LLM (Large Language Model) pré-entraînés.

Fonctionnement : Le système extrait automatiquement les données du portail Data.gouv, les traite, les stocke dans une base de données vectorielle pour une recherche sémantique efficace. Lorsqu'un utilisateur pose une question, le système la transforme en requête vectorielle, récupère les documents ou fragments de données les plus pertinents, puis transmet ces informations enrichies à un LLM. Ce dernier génère alors une réponse claire et contextualisée à la requête de l'utilisateur.

Modèle de Décision Médicale à partir de Bilans Biologiques

Projet Personnel / Académique

Ce projet visait à analyser des données de bilans d'examens biologiques pour concevoir un modèle capable de recommander des décisions médicales (par exemple, s'il est nécessaire ou non de prescrire un traitement ou de faire des examens complémentaires).

Technologies utilisées : Python, Pandas, Scikit-learn, Machine Learning (Classification).

Approche : Les données brutes des bilans ont été nettoyées et prétraitées. Des algorithmes d'apprentissage supervisé (comme Random Forest ou XGBoost) ont été entraînés sur un ensemble de données étiquetées (bilan -> décision). Le modèle final est capable d'analyser un nouveau bilan et de prédire la probabilité de différentes actions médicales, fournissant un outil d'aide à la décision pour les professionnels de santé.
